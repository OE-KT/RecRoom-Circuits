Iterates over the input list. The loop exec fires for each element in the list. The Done exec fires once the end of the list has been reached.

| Input Name | Input Type |
|-----------|-----------|
| *No name.* | exec |
| Items | List&ltany&gt |

| Output Name | Output Type |
|-----------|-----------|
| Loop | exec |
| Item | any |
| Done | exec |
